# Vue Weather Application

![](screenshot.png)

## Getting Started

Change `.env.example` to `.env` and update value of [Open Weather](https://openweathermap.org/) and [Here Map](https://developer.here.com/)

```
VITE_OPENWEATHER_API_KEY=
VITE_HERE_API_KEY=
```

Running in localhost:5000

```
$ pnpm dev
```

## Materials
